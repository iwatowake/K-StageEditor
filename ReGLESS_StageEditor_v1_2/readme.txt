/ =====================================
  リグレスの匣　ステージエディターv1.2
/ =====================================

*　ショートカット一覧
　 w,r … レイヤ選択
　 e　 … 選択中のレイヤの表示/非表示切替
　 s,f … 選択中のレイヤを回転
　 x,v … 設置物選択
 　Shift+メインレイヤ内の設置物をクリック … 設置物を削除

*　セーブ方法
　 エリアを指定し、最短手数(C-Path)とステージ名(StageName)を入力してSaveボタンを押して下さい。
　 "ReGLESS_StageEditor_vx_x_Data/StageData/ステージ名" のフォルダが生成され、
 　その中にスクリーンショットとtxtファイルが生成されます。

*　ロード方法
　 StageDataフォルダ内にステージデータがあれば読み込むことができます。
　 読み込みたいステージ名を入力してLoadボタンを押して下さい。

　 ※読み込んだステージを編集した後、同じ名前で保存すると古いデータが上書きされるので注意して下さい。
